## Project

SG Bus Live — a native SwiftUI **iPhone-only** app showing live Singapore bus arrivals from
LTA DataMall. Bundle id `com.alfredang.sgbus`, ASC app id `6782321279`, team `GU9WTSTX9M`.

## Build & run

The Xcode project is **generated by XcodeGen** — edit `project.yml`, never `SGBus.xcodeproj/project.pbxproj` directly (it is regenerated and overwritten).

```bash
xcodegen generate                      # regenerate SGBus.xcodeproj after editing project.yml
# Run on a simulator:
xcodebuild -project SGBus.xcodeproj -scheme SGBus -configuration Debug \
  -sdk iphonesimulator -destination 'platform=iOS Simulator,name=iPhone 17 Pro Max' build
# Build + install on a physical device (hardware UDID from `xcrun xctrace list devices`):
xcodebuild -project SGBus.xcodeproj -scheme SGBus -configuration Debug \
  -destination "id=<HARDWARE_UDID>" -allowProvisioningUpdates -derivedDataPath build-device build
xcrun devicectl device install app --device <UDID> "build-device/Build/Products/Debug-iphoneos/SG Bus.app"
```

There is **no test target** in this project.

### Required local config (not in git)

The app reads the LTA DataMall key from `Info.plist` key `LTAAccountKey`, injected at build
time from `Config/Local.xcconfig` (`LTA_ACCOUNT_KEY = ...`). Without it, arrival/bus-stop calls
fail. Copy `Config/Local.xcconfig.example` → `Config/Local.xcconfig` and fill the key. This file
is gitignored.

## Architecture

SwiftUI + MVVM, async/await, no third-party dependencies.

- `Views/ContentView.swift` — the single screen: search field (postal code / bus stop ID),
  "Use my location", nearby stops, and the selected stop's arrivals with bus-number filter chips.
- `ViewModels/BusArrivalsViewModel.swift` (`@MainActor`) — the orchestrator. It **owns**
  `LocationManager`, `BusStopStore`, `LTADataMallClient`, and `OneMapClient`, and re-publishes
  the nested observable objects' changes via Combine (`objectWillChange` forwarding) so the view
  only observes the view model. Auto-requests location on first `onAppear`.
- `Services/LocationManager.swift` — CoreLocation one-shot GPS fix. Delegate callbacks are
  `nonisolated` and hop to `@MainActor`; inside those hops use `self.manager` (not the delegate's
  `manager` parameter) to satisfy Swift 6 strict concurrency.
- `Services/BusStopStore.swift` — downloads the **entire** LTA bus-stop directory once
  (`fetchAllBusStops`, paginated by `$skip` 500 at a time), caches it to `Caches/busstops.json`,
  and answers nearest-by-GPS (haversine via `CLLocation.distance`), exact-code lookup, and text
  search.
- `Services/OneMapClient.swift` — geocodes a 6-digit Singapore postal code to a coordinate via
  OneMap's public `elastic/search` endpoint (no auth).
- `Services/LTADataMallClient.swift` — LTA DataMall calls (`v3/BusArrival`, `BusStops`); sends
  the `AccountKey` header.

Search routing in the view model: 5-digit numeric → bus stop ID lookup; 6-digit numeric →
postal-code geocode → nearest stops; otherwise free-text stop search.

## App Store / distribution

Release signing is **manual** (`Apple Distribution` + profile `SG Bus App Store Local`) per the
Release config in `project.yml`; Debug uses automatic signing. Bump `CURRENT_PROJECT_VERSION`
(build number) in `project.yml` on every upload.

`scripts/` automates App Store Connect via the API (loads `.env`, mints a JWT with
`asc_jwt.swift`):

```bash
set -a; source .env; set +a
python3 scripts/asc_submit.py status                     # app/version/build state
python3 scripts/asc_submit.py set-metadata              # copyright, URLs, description, keywords, subtitle, whatsNew
python3 scripts/asc_submit.py attach-build --build N
python3 scripts/asc_submit.py screenshots --type APP_IPHONE_67 a.png b.png
python3 scripts/asc_submit.py submit
swift scripts/make_app_icon.swift out.png               # regenerate the 1024px bus app icon
```

`.env` holds ASC credentials + marketing copy (gitignored). The detailed end-to-end submission
playbook and gotchas live in `.claude/skills/app-store-submission/SKILL.md`.

### Non-obvious gotchas (each has bitten this app)

- **Top-level `CFBundleIconName`** must be present in Info.plist (`project.yml` sets it). With
  only the nested `CFBundleIcons` key, a build uploads "successfully" then silently fails ASC
  processing and never appears.
- **Globally-unique names**: both the App Store listing name and `CFBundleDisplayName` must be
  unique. "SG Bus" was taken → the app is "SG Bus Live".
- **iPhone-only**: `TARGETED_DEVICE_FAMILY = 1`. If a universal build is left `VALID`, ASC
  re-demands iPad screenshots — expire stale universal builds.
- **App Store "What's New" (`whatsNew`)** can only be set on an *update* of an already-released
  version; it is rejected on a first, never-released version. See `CHANGELOG.md`.
- App Privacy label, age rating, pricing, and availability are **UI-only** (no public API) —
  done via App Store Connect / Playwright.
